## Overview

Trenova provides a built-in hazardous materials catalog and enforcement engine that keeps your operations in compliance with 49 CFR DOT regulations. You maintain a library of hazmat records, define co-loading restrictions with segregation rules, and link specific commodities to hazmat designations so every shipment carrying dangerous goods is flagged and controlled automatically.

---

## Hazardous materials catalog

The **hazardous materials catalog** is your authoritative reference for every hazardous substance your organization moves. Each record maps to a specific DOT entry and carries the information drivers, shippers, and emergency responders need.

| Field | Description |
|---|---|
| **Code** | Short internal identifier, up to 10 characters |
| **Name** | Common or trade name of the material, up to 100 characters ‚Äî required |
| **Description** | Detailed description ‚Äî required |
| **Status** | `Active` or `Inactive` |
| **UN Number** | 4-digit UN/NA identification number (e.g., `1993`) |
| **Proper Shipping Name** | DOT-required name as listed in 49 CFR Part 172 |
| **Class** | Primary hazard class ‚Äî see class table below |
| **Packing Group** | DOT packing group indicating relative hazard level |
| **Subsidiary Hazard Class** | Secondary hazard classification (up to 20 characters) |
| **ERG Guide Number** | Emergency Response Guidebook section number |
| **Label Codes** | Required label codes (up to 50 characters) |
| **Special Provisions** | Applicable 49 CFR special provisions, comma-separated |
| **Handling Instructions** | Specific loading, unloading, or transport instructions |
| **Emergency Contact** | Name or organization for emergency response queries |
| **Emergency Contact Phone** | 24-hour emergency contact phone number |
| **Quantity Threshold** | Reportable quantity threshold (required when `IsReportableQuantity` is true) |
| **Placard Required** | Whether DOT placarding is required |
| **Is Reportable Quantity** | Whether releases trigger CERCLA/EPCRA reporting |
| **Marine Pollutant** | Whether the material is classified as a marine pollutant |
| **Inhalation Hazard** | Whether the material poses an inhalation risk |

## Segregation rules

**Hazmat segregation rules** define which hazard classes and specific materials cannot be co-loaded on the same trailer. Trenova evaluates these rules during assignment to flag or block loads that would violate DOT segregation requirements.

| Field | Description |
|---|---|
| **Name** | Rule name, up to 100 characters ‚Äî required |
| **Description** | Explanation of what this rule covers |
| **Status** | `Active` or `Inactive` |
| **Class A** | First hazard class in the pairing ‚Äî required |
| **Class B** | Second hazard class in the pairing ‚Äî required |
| **Segregation Type** | The enforcement action ‚Äî see type table below |
| **Hazmat A** | Optional: restrict the rule to a specific material within Class A |
| **Hazmat B** | Optional: restrict the rule to a specific material within Class B |
| **Minimum Distance** | Required separation distance (when type is `Distance`) |
| **Distance Unit** | `FT`, `M`, `IN`, or `CM` |
| **Has Exceptions** | Whether any exceptions to this rule apply |
| **Exception Notes** | Required when `HasExceptions` is true (5‚Äì1,000 characters) |
| **Reference Code** | Internal or regulatory reference code for this rule |
| **Regulation Source** | Cite the regulation this rule implements (e.g., `49 CFR 177.848`) |

### Segregation types

| Type | Enforcement |
|---|---|
| `Prohibited` | These classes may never be loaded together ‚Äî hard block |
| `Separated` | Must be kept apart within the trailer (e.g., separated by cargo) |
| `Distance` | Must maintain a minimum physical distance apart |
| `Barrier` | Must be separated by a physical barrier or bulkhead |

<Note>
When `Class A` and `Class B` are the same hazard class, the rule must target two different specific material records via `HazmatAID` and `HazmatBID`. Trenova enforces this at save time to prevent ambiguous rules.
</Note>

## Commodity-level hazmat designation

You designate a commodity as hazardous by linking it to a hazardous material record in the commodity catalog. When that commodity appears on a shipment via `ShipmentCommodity`, Trenova automatically:

1. Applies the hazmat flag to the shipment
2. Triggers driver endorsement validation (drivers must hold an `H` or `X` CDL endorsement)
3. Evaluates active segregation rules against all other commodities on the load
4. Enforces any applicable temperature range requirements

---

## Temperature requirements

For temperature-sensitive hazardous materials ‚Äî such as certain organic peroxides or biological substances ‚Äî you can specify a temperature range on the shipment itself using the `TemperatureMin` and `TemperatureMax` fields (in ¬∞F). Dispatch can use this to select appropriate refrigerated equipment types.

---

## Driver hazmat certification

Drivers transporting hazmat must hold a valid CDL hazmat endorsement. Trenova checks the worker profile's `Endorsement` field and `HazmatExpiry` date during assignment:

- Endorsement `H` ‚Äî hazardous materials only
- Endorsement `X` ‚Äî hazardous materials + tanker combined

Both require a non-expired `HazmatExpiry` date on the worker profile. Trenova blocks assignment if the expiry is missing or past.
